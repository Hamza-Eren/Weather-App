# Weather-App
Hava durumu uygulaması

Yazıldığı dil : Python

### Kullanılan kütüphaneler
* tkinter
* PIL
* requests
* bs4
* json

### Açıklama:
Bu uygulama animasyonlu bir şekilde hazırlanmıştır ve openweathermap API'ı kullanarak yazılmıştır.
